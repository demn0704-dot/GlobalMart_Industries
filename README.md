# 🚀 SAP S/4HANA ERP Implementation — GlobalMart Industries

## 📌 Project Summary
Designed and documented an end-to-end **SAP S/4HANA ERP implementation** for a multi-location enterprise, covering **FI, MM, and SD modules** with fully integrated **Procure-to-Pay (P2P)** and **Order-to-Cash (O2C)** cycles.

✔ Focus on real-world business processes, GST compliance, and enterprise-level configuration.

---

## 🎯 Key Highlights (Resume Focus)
- Implemented **FI, MM, SD modules** with cross-module integration  
- Designed complete **P2P and O2C workflows**  
- Configured **GST (CGST, SGST, IGST), TDS, TCS**  
- Applied **3-way matching (PO–GR–Invoice)** in MM  
- Built **pricing procedures & credit management** in SD  
- Defined **organizational structure (Company Code, Plant, Sales Org)**  
- Created **custom reports (ABAP-based)** for business insights  
- Followed **SAP Activate methodology (Agile-based implementation)**  

---

## 🧩 Modules & Responsibilities

### 💰 Financial Accounting (FI)
- Configured G/L, AP, AR, Asset Accounting  
- Set up **parallel ledgers (Ind AS, IFRS)**  
- Automated payments using **F110**  
- Implemented taxation and compliance  

### 📦 Materials Management (MM)
- Configured procurement cycle (PR → PO → GR → Invoice)  
- Set up **inventory management & valuation (OBYC)**  
- Implemented **release strategy (multi-level approval)**  
- Enabled **3-way invoice verification (MIRO)**  

### 🛒 Sales & Distribution (SD)
- Configured **sales order to billing cycle**  
- Designed **pricing procedure (ZVKWRT)**  
- Implemented **ATP (availability check)**  
- Set up **credit management & billing integration**  

---

## 🔄 End-to-End Process Coverage

### 📥 Procure-to-Pay (P2P)
PR → RFQ → PO → Goods Receipt → Invoice → Payment  

### 📤 Order-to-Cash (O2C)
Sales Order → Delivery → PGI → Billing → Payment
